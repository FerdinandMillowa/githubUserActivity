# GitHub User Activity CLI

A professional Command Line Interface (CLI) tool that fetches and displays real-time activity for any GitHub user. Built as the second project in my **Backend Development Roadmap**.

* **JSON Parsing:** Navigating complex, nested JSON data structures.
* **Error Handling:** Managing API-specific errors like `404 Not Found` (invalid users) and connection issues.
* **Zero Dependencies:** Built entirely using Python's standard library.
